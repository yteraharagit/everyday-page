GitHubの使い方でもまとめてみる

SourceTree

Mac を使用している関係で全て Mac ベースとなるが、CLI での操作で developer tools をインストールするのがなんとなく嫌だったのと、GUI っ子なので、 SourceTree を入れて簡単に操作出来るようにしたかった。

SourceTreeのインストール

インストール方法は簡単、ただ Atlassian アカウントが無い人は作成する必要がある

リポジトリ

ローカルとリモートリポジトリがあり、クローンさせる事で同期を取らせるみたい

ここらへんは GUI であれば直感的に操作可能で、つまずく事はないだろう

GitHub Pages

何と言っても、これ。GitHub Pages が今回使いたいがために、GitHub を今回使い始めたのだ。

つまずいたところとしては、リポジトリ内にある index.html のみしかなぜか表示されなかった事。いろいろ調べていると index 意外で表示させている方はたくさんいたのに、なぜか表示出来なかった。。。

ノンテクのデザイナーの人が 180 日間毎日 html/css/js を駆使してページを公開しているのを見て、今回これを始め、そーいう感じでやりたかったが、リポジトリ内でディレクトリを切るとリンクが飛ばなかったので、リポジトリ内にだらだらとファイルを増やす事になる。。。
